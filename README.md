# Services Visas US

Pagina web profesional creada con Next.js, React, HTML y CSS para una agencia
de asesoria de viajes y visas.

## Ejecutar el proyecto

Instala Node.js y luego ejecuta:

```bash
npm install
npm run dev
```

Abre `http://localhost:3000` en el navegador.

## Ajustar WhatsApp

Edita el numero y el mensaje en:

```txt
components/siteConfig.js
```

La constante principal es `WHATSAPP_NUMBER`.

## Logo

El logo principal esta preparado en:

```txt
public/logo.png
```
